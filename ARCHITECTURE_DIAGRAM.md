## Data Flow

```
Price Feed → Scanner → Validator → Risk → Executor → Settlement → P&L
```

## Redis Pub/Sub Topics

```
┌─────────────────────────────────────────────────────────────────────┐
│                         Redis Pub/Sub                               │
└─────────────────────────────────────────────────────────────────────┘

Topics:
┌─────────────────────────────────────────────────────────────────────┐
│  trading:opportunities  ───► Scanner publishes, others subscribe   │
├─────────────────────────────────────────────────────────────────────┤
│  trading:validation_result  ───► Validator publishes, Risk subscribes│
├─────────────────────────────────────────────────────────────────────┤
│  trading:risk  ───► Risk Manager publishes, Executor subscribes    │
├─────────────────────────────────────────────────────────────────────┤
│  trading:execution  ───► Executor publishes, Reconciler subscribes  │
├─────────────────────────────────────────────────────────────────────┤
│  trading:reconciliation ───► Reconciler publishes (reports)        │
└─────────────────────────────────────────────────────────────────────┘

Channels:
┌─────────────────────────────────────────────────────────────────────┐
│  trading:opportunities  ───► All agents can subscribe               │
├─────────────────────────────────────────────────────────────────────┤
│  trading:alerts  ───► Emergency stop notifications                  │
├─────────────────────────────────────────────────────────────────────┤
│  trading:stop_all  ───► Kill signal to all agents                   │
└─────────────────────────────────────────────────────────────────────┘
```

│                    Scaling Strategy                                 │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│  Scanner     │  Validator  │  Risk    │  Executor  │  Reconciler   │
│  (1)         │  (1)        │  (1)      │  (1)        │  (1)           │

┌─────────────────────────────────────────────────────────────────────┐
